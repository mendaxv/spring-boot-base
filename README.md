spring boot基础项目搭建脚手架，目前添加了如下功能：
- 统一返回：自定义返回json格式
- 统一异常处理：全局异常捕获、自定义异常
- 日志切面：打印请求入参、请求url、请求Method、请求IP等
- 跨域解决方案
- 参数校验
- 整合logback
